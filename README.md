## Featured Projects

### Neural Network from Scratch
**Location:** [`ai-ml/neural-network-from-scratch/`](./ai-ml/neural-network-from-scratch/)  
**Inspiration:** Sentdex's Neural Networks from Scratch series

Neural network implementation using **pure NumPy** - no high-level ML frameworks. Demonstrates understanding of machine learning fundamentals through manual forward pass implementation.

**Technical Implementation:**
- Configurable dense layers with forward propagation using matrix multiplication
- Categorical Cross-Entropy loss with numerical stability optimizations

**Key Skills:**
- Linear algebra and NumPy vectorization
- Neural network architecture and modular design

---

### Pong Game Engine
**Location:** [`games/pong/`](./games/pong/)

Fully-featured recreation of classic Pong with advanced game mechanics and physics simulation.

**Technical Implementation:**
- Custom timer-based game loop with ~100 FPS frame rate
- Real-time collision detection with paddle and boundary interactions

**Key Skills:**
- Event-driven programming and real-time systems
- Game development architecture

### Data Structures Library
**Location:** [`data-structures/`](./data-structures/)

Production-quality implementations of fundamental data structures with full Python integration and comprehensive testing.

**Stack (Array-Based):**

**Technical Implementation:**

### System-Level Programming with Python
